# Solo Movie Watchlist 

Simple project to demonstrate no frills javascript/html/css skills and use of OMDb api

Movie watchlist using api: https://www.omdbapi.com/

# how to use
1) Download project

2) In browser, navigate to downloaded file folder and open the index.html file
